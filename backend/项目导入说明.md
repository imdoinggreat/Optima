# 项目信息批量导入指南

不用一个个复制粘贴，用 CSV 批量导入硕士项目到系统。

## 一、已有爬取数据（最快）

若你已有 `crawler/data/programlist_crawled.csv`（crawler 跑过后的产出）：

```bash
cd backend
./venv/bin/python import_programs.py crawler/data/programlist_crawled.csv
```

会自动过滤掉本科项目、首页等噪音，只导入硕士项目。

---

## 二、自己整理 CSV 再导入

### 步骤 1：生成模板

```bash
cd backend
./venv/bin/python import_programs.py --template
```

会生成 `programs_import_template.csv`。

### 步骤 2：填数据（三种方式）

| 方式 | 说明 |
|------|------|
| **从网页复制表格** | 打开学校官网项目列表，全选表格 → 粘贴到 Excel → 调整列名为 program_name, university, program_website 等 → 另存为 CSV |
| **从现有 Excel 合并** | 多个来源的表格在 Excel 里合并、删除重复、统一列名 → 另存为 CSV |
| **手动填写** | 按模板逐行填，必填仅 program_name + university，其余可选 |

### 步骤 3：导入

```bash
./venv/bin/python import_programs.py 你的文件.csv
```

---

## 三、CSV 格式要求

### 格式 A：简化版（推荐）

| 列名 | 必填 | 说明 |
|------|:---:|------|
| program_name | ✓ | 项目全称 |
| university | ✓ | 学校名 |
| program_website | | 官网链接 |
| degree_subtype | | MA/MS/MPP/MPA/MBA 等 |
| duration_months | | 学制（月） |
| total_cost_usd | | 总学费（美元） |
| is_stem | | 1/0 或 true/false |
| prerequisite_courses | | 用 `|` 分隔，如：微积分\|统计学 |

### 格式 B：programlist 兼容

| 列名 | 说明 |
|------|------|
| university | 学校 |
| program | 项目名 |
| degree | 学位类型 |
| school | 所属学院 |
| official_url | 官网链接 |
| field | 领域 |

---

## 四、常见参数

```bash
# 预览，不写入数据库
./venv/bin/python import_programs.py 文件.csv --dry-run

# 导入全部行（包括本科、首页等，默认会过滤）
./venv/bin/python import_programs.py 文件.csv --all

# 不跳过已存在的项目（默认会跳过重复）
./venv/bin/python import_programs.py 文件.csv --no-skip-dup
```

---

## 五、快速整理技巧

1. **Excel 批量操作**：从多个网页复制 → 粘贴到同一 Excel 表 → 用「数据」→「分列」统一格式 → 用 VLOOKUP 补充缺失列  
2. **先粗后精**：第一版只填 program_name、university、program_website，导入后再慢慢在系统里补充详情  
3. **复用爬虫产出**：若有 programlist.xlsx，可跑 crawler 得到 programlist_crawled.csv，再直接导入
