# 美国硕士项目推荐核心维度全表

以下维度**100%贴合北美顶尖院校官方招生规则、国际生就读核心诉求、选校测评系统的量化落地需求**，覆盖从「准入门槛→学术价值→职业出口→就读体验→风险兜底」的全周期，同时区分通用权重与分项目类型的差异化优先级，可直接匹配选校匹配公式与32个文商科专业变量体系。

---

## 一、维度总框架与通用权重优先级

按国际生选校的核心诉求与录取逻辑，分为5大类，通用权重适配绝大多数授课型/职业型硕士（占美国硕士申请的90%以上），研究型硕士可单独调整权重。

| 大类维度 | 通用权重 | 核心定位 |
| :---: | :---: | :--- |
| 一、核心申请适配维度 | 30% | 准入门槛+录取概率核心，是项目推荐的基础前提 |
| 二、职业发展与出口维度 | 30% | 硕士就读的核心终局诉求，决定项目的长期价值 |
| 三、学术培养与资源维度 | 20% | 项目含金量的核心支撑，决定就读的核心收获 |
| 四、就读体验与个性化适配维度 | 12% | 决定就读幸福感、毕业成功率与长期适配度 |
| 五、成本与风险控制维度 | 8% | 兜底项，决定投入产出比与就读稳定性 |

---

## 二、细分维度全解（可量化、可落地、可直接用于测评打分）

### 一、核心申请适配维度（30%）

**核心作用**：先筛掉「申请者够不到、录取概率极低」的无效项目，再匹配「项目偏好与申请者特质高度契合」的高概率项目，是推荐的第一道门槛。

#### 1. 硬性准入门槛（权重15%，一票否决项）

| 细分维度 | 核心说明 | 关键考察点 |
| :---: | :--- | :--- |
| 学术背景门槛 | 项目对本科背景的硬性要求，是跨专业申请的核心红线 | 本科专业限制、最低GPA要求、强制先修课程（如BA要求Python/统计、金工要求高数/线代） |
| 标化成绩门槛 | 国际生必须满足的语言/标化底线 | 托福/雅思最低分、GRE/GMAT强制/可选政策、项目历年录取平均标化 |
| 工作经验门槛 | 项目对全职工作的偏好/要求，是应届生/在职申请者的核心筛选项 | 强制工作年限要求、偏好应届生还是在职申请者、平均录取工作年限 |

#### 2. 录取竞争与概率维度（权重7%）

| 细分维度 | 核心说明 | 关键考察点 |
| :---: | :--- | :--- |
| 整体录取难度 | 项目的整体竞争激烈程度 | 历年整体录取率、国际生录取率、中国学生录取占比 |
| 背景友好度 | 项目对特定背景申请者的包容度 | 跨专业友好度、低GPA友好度、无工作经验友好度、陆本/海本偏好 |

#### 3. 软匹配度（权重8%，录取概率的核心增量）

完全匹配「兴趣-特质-项目」匹配模型，是区分同质化申请者的核心。

| 细分维度 | 核心说明 | 关键考察点 |
| :---: | :--- | :--- |
| 项目使命与申请者目标匹配度 | 项目的核心培养定位，与申请者的职业/学术目标的契合度 | 项目培养学术研究者还是职场从业者、培养公益领袖还是商业精英、细分培养方向与申请者兴趣的重合度 |
| 项目文化与申请者特质匹配度 | 项目的整体氛围，与申请者的行为特质、学习风格的契合度 | 竞争型还是协作型氛围、结构化培养还是灵活定制化、理论导向还是实践导向 |

### 二、职业发展与出口维度（30%，硕士申请的核心终局）

**核心作用**：绝大多数国际生读美国硕士的核心目标是就业，该维度直接决定项目的投入产出比，是推荐排序的核心依据。

#### 1. 硬核就业数据（权重18%，核心硬指标）

| 细分维度 | 核心说明 | 关键考察点 |
| :---: | :--- | :--- |
| 就业率 | 国际生专属就业率远重于整体就业率 | 毕业3/6/12个月整体就业率、国际生就业率、留美vs回国就业比例 |
| 薪资水平 | 项目的真实就业回报 | 毕业起薪中位数、平均薪资、行业薪资分布、薪资分位数 |
| 就业去向 | 项目的真实就业赛道，与申请者职业目标的匹配度 | 核心就业行业、标杆雇主、核心对口岗位、行业头部企业的录取人数 |
| 地理位置就业红利 | 项目所在地的就业市场便利性 | 所在城市是否为对应行业的核心聚集地（如金融-纽约、科技-旧金山、政策-华盛顿DC）、本地企业校招偏好、实习通勤便利性 |

#### 2. 职业发展配套资源（权重8%）

| 细分维度 | 核心说明 | 关键考察点 |
| :---: | :--- | :--- |
| 专属职业服务 | 项目对就业的支持力度 | 项目专属职业中心vs全校通用服务、行业专属职业导师、简历修改/模拟面试/内推服务 |
| 实习与校招资源 | 项目对接企业的能力 | 强制实习学分要求、校企合作实习项目、企业校招宣讲会、暑期实习内推通道 |
| 留美签证支持 | 国际生留美的核心保障 | OPT申请指导、H1B sponsorship支持、国际生专属留美就业辅导 |

#### 3. 校友网络资源（权重4%）

| 细分维度 | 核心说明 | 关键考察点 |
| :---: | :--- | :--- |
| 校友分布 | 校友在目标行业的话语权 | 核心行业/核心城市的校友集中度、头部企业的校友任职情况 |
| 校友活跃度 | 校友网络的实际价值 | 校友导师计划、校友内推机制、线下校友社群、行业分享会频次 |

### 三、学术培养与资源维度（20%，项目含金量的核心支撑）

**核心作用**：决定申请者就读期间能学到什么、获得什么核心能力，是长期职业发展的基础，研究型硕士权重可提升至40%以上。

#### 1. 课程体系与培养定位（权重10%）

| 细分维度 | 核心说明 | 关键考察点 |
| :---: | :--- | :--- |
| 核心培养轨道 | 项目的核心定位，与申请者目标匹配 | STEM认证（36个月OPT）/非STEM、研究型（Thesis）/授课型（Coursework/Capstone）、通用型/细分方向专精 |
| 课程灵活性与深度 | 项目的定制化空间 | 选修课占比、跨学院/跨专业选课权限、双学位/辅修机会、细分赛道的课程覆盖度 |
| 实践学术配套 | 项目的落地能力 | Capstone顶点项目、行业案例竞赛、模拟项目（如模拟国会/并购谈判）、实地调研/田野调查机会 |
| 跨专业友好度 | 对跨专业申请者的支持 | 桥梁课程、暑期预科、基础先修课补课通道、零基础友好的课程设置 |

#### 2. 师资与学术资源（权重7%）

| 细分维度 | 核心说明 | 关键考察点 |
| :---: | :--- | :--- |
| 师资配置 | 授课老师的背景与可用性 | 终身教授比例、业界导师比例、教授研究方向与申请者兴趣的匹配度、师生比、教授是否带硕士生做科研/项目 |
| 学术硬件资源 | 项目的学术支撑能力 | 专属研究中心、实验室、行业数据库、图书馆资源、学术会议支持、论文发表机会 |

#### 3. 校企/行业联动资源（权重3%）

| 细分维度 | 核心说明 | 关键考察点 |
| :---: | :--- | :--- |
| 行业合作 | 项目与业界的联动深度 | 行业客座讲师、企业合作项目、行业workshop、高管导师计划 |

### 四、就读体验与个性化适配维度（12%）

**核心作用**：决定申请者就读期间的幸福感、心理健康、毕业成功率，匹配「兴趣-校园社区」适配模型。

#### 1. 学制与就读节奏（权重4%）

| 细分维度 | 核心说明 | 关键考察点 |
| :---: | :--- | :--- |
| 学制时长 | 与申请者时间规划、实习需求的匹配度 | 1年制/1.5年制/2年制（2年制可做暑期实习，1年制无完整暑期实习窗口） |
| 学期制度与节奏 | 与申请者学习风格的匹配度 | Semester制（节奏平缓）/Quarter制（节奏快、压力大）、考核方式、毕业难度、历年毕业率/辍学率 |

#### 2. 班级与社群氛围（权重3%）

| 细分维度 | 核心说明 | 关键考察点 |
| :---: | :--- | :--- |
| 项目规模 | 就读体验的核心影响因素 | 每年招生人数、班级大小、师生比、小班授课占比 |
| 班级多元化 | 社群适配度 | 国际生比例、中国学生比例、学生背景多元化程度、协作型/竞争型班级氛围 |

#### 3. 地理位置与生活配套（权重3%）

| 细分维度 | 核心说明 | 关键考察点 |
| :---: | :--- | :--- |
| 生活成本 | 就读的经济压力 | 所在城市房租、日常消费、交通成本 |
| 安全与配套 | 就读的基础保障 | 校区/所在城市治安情况、华人配套、交通便利性 |

#### 4. 学生支持体系（权重2%）

| 细分维度 | 核心说明 | 关键考察点 |
| :---: | :--- | :--- |
| 国际生支持 | 国际生适配的核心保障 | 国际学生办公室、语言支持、文化适应辅导、签证服务 |
| 学业与心理健康支持 | 就读稳定性的保障 | 学业辅导、心理咨询、压力管理服务、弱势群体帮扶资源 |
| 课外兴趣资源 | 与申请者个人爱好的匹配度 | 学生社团、研究小组、志愿机会、领导力项目、兴趣社群的丰富度 |

---

### 五、成本与风险控制维度（8%，兜底项）

**核心作用**：规避高风险、低性价比的项目，保障就读的稳定性与投入产出比。

#### 1. 经济成本与资金支持（权重5%）

| 细分维度 | 核心说明 | 关键考察点 |
| :---: | :--- | :--- |
| 总就读成本 | 项目的整体投入 | 学费、学杂费、生活费、保险费的总预算 |
| 国际生资助机会 | 降低成本的核心渠道 | 国际生可申请的奖学金、助学金、TA/RA助教助研机会、学费减免、生活 stipend |
| 投入产出比ROI | 项目的性价比 | 总投入vs毕业起薪、薪资回本周期 |

#### 2. 政策与项目风险（权重3%）

| 细分维度 | 核心说明 | 关键考察点 |
| :---: | :--- | :--- |
| STEM认证 | 国际生留美的核心红线 | 是否具备美国国土安全局认证的STEM代码，36个月OPT资格 |
| 项目稳定性 | 规避踩坑的核心 | 项目办学时长、招生规模稳定性、是否有停办/缩招风险、官方认证资质 |
| 签证风险 | 就读的前提保障 | 项目类型对应的签证通过率、敏感专业风险 |

## 三、分项目类型的权重调整表

适配32个文商科细分方向，不同类型项目的核心考察维度权重可灵活调整，精准匹配项目定位。

| 项目类型 | 核心权重调整 | 核心推荐逻辑 |
| :---: | :--- | :--- |
| 商科职业型硕士（MSF/BA/市场营销/酒店管理） | 职业发展维度权重提升至40%，申请适配维度25%，学术培养15% | 核心看就业出口、留美机会、校企资源、STEM认证 |
| 公共事务类硕士（MPP/MPA/社会工作） | 申请适配维度35%，职业发展25%，学术培养20% | 核心看项目使命与申请者价值观匹配度、政府/国际组织合作资源、政策实践机会 |
| 法律/合规类硕士（LLM/合规管理） | 申请适配维度35%，职业发展30%，学术培养20% | 核心看bar考试通过率、律所合作资源、行业导师配置、先修课门槛 |
| 传媒/艺术类硕士（IMC/新闻/数字媒体） | 职业发展维度35%，学术培养25%，申请适配20% | 核心看行业实践资源、校企合作、作品集/项目支持、地理位置（LA/纽约） |
| 研究型/申博导向硕士（经济学/教育学/人类学） | 学术培养维度权重提升至45%，申请适配30%，职业发展10% | 核心看导师研究方向匹配度、科研资源、论文发表支持、博士升学率 |
| 跨学科硕士（创新创业/数字人文/跨学科研究） | 申请适配维度35%，学术培养25%，职业发展20% | 核心看课程灵活性、跨学院选课权限、定制化培养空间、与申请者多元兴趣的匹配度 |

## 四、与现有系统 mapping 参考

| 本框架维度 | 现有 `matching.py` 对应 |
| :--- | :--- |
| 核心申请适配（准入门槛） | `academic_fit`、`prerequisite_fit` |
| 职业发展与出口 | `career_fit`、`employment_rate_3month`、`capstone_required` |
| 学术培养 | `course_flexibility`、`cross_registration` |
| 就读体验 | `cohort_size`、`preference_weights`（small_cohort、interdisciplinary） |
| 成本与风险 | `total_cost_usd`、`is_stem` |
| 软匹配度 | `assessment_fit`（人格 + 选校偏好） |
